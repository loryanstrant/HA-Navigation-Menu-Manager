# Navigation Menu Manager

A Home Assistant integration that gives you a single place to define dashboard navigation menus, then drop a custom **Navigation Menu** card on any view to render them.

- Edit menu items once — every dashboard updates live
- Active-view highlighting based on the browser URL
- Custom Lovelace card with preview in the card picker
- Sidebar admin panel for managing menus
- HACS-ready

After install, restart Home Assistant, then add the **Navigation Menu Manager** integration via *Settings → Devices & Services → Add Integration*. A **Nav Menus** entry will appear in your sidebar.
